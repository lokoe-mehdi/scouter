DeadlockRetry

trait DeadlockRetry

Trait pour gérer les deadlocks PostgreSQL avec retry et backoff exponentiel

Ce trait fournit des méthodes pour exécuter des requêtes SQL avec un mécanisme de retry automatique en cas de deadlock (SQLSTATE 40P01).

Utilisation :

class MyClass {
    use DeadlockRetry;

    public function doSomething() {
        $this->executeWithRetry($pdo, function($pdo) {
            // Code qui peut causer un deadlock
        });
    }
}
