JsRenderer

Since: 2.0.0

class JsRenderer

Client pour le microservice de rendu JavaScript

Cette classe communique avec un service Puppeteer (Node.js) qui exécute le JavaScript des pages et retourne le HTML rendu.

Utilisé pour crawler les SPA (Single Page Applications) et les sites avec contenu généré dynamiquement côté client.

Architecture :

Scouter -> JsRenderer -> Puppeteer (Docker) -> Site web
