Simhash

class Simhash

Détection de contenu dupliqué via Simhash

Implémentation de l'algorithme Simhash pour détecter le contenu dupliqué ou quasi-dupliqué (near-duplicate) entre les pages.

Le simhash est une empreinte 64-bit où des documents similaires ont des hashes similaires. La distance de Hamming entre deux simhashes indique le degré de similarité.

Seuils de similarité :

  • Distance 0 : Contenu identique
  • Distance 1-3 : Near-duplicate (très similaire)
  • Distance 4-10 : Contenu lié (similaire)
  • Distance >10 : Contenu différent
